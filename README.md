# react-scad minimal example

A minimal [react-scad](https://github.com/react-scad/react-scad) project: one cube, one sphere, unioned and written to `model.scad`.

## Prerequisites

- **Node.js** 18+
- **npm** (or pnpm / yarn)

## Setup

```bash
npm install
```

## Run

```bash
npm start
```
